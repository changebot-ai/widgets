Changebot Widget - Production Example

This is a simple example showing the widget in action with real production data.

Your Application Header

The badge above shows the number of new updates from your Changebot feed. It will:

  • Poll for updates every 30 seconds
  • Show the count of unread updates
  • Remember what you've already seen using localStorage
  • Hide automatically when there are no new updates

Integration Code

<!-- Add once per page --> <changebot-provider slug="changebot" poll-interval="30"></changebot-provider> <!-- Add badge wherever you want it to appear --> <changebot-badge></changebot-badge>
