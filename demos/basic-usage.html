This example demonstrates the most basic setup for Changebot Widgets:

  • One provider component that manages state
  • One badge component to show update counts
  • One drawer component to display updates

The components communicate automatically through the provider using custom events.

<changebot-provider slug="demo-team" scope="default"> <changebot-badge scope="default"></changebot-badge> <changebot-panel scope="default" mode="drawer-right"></changebot-panel> </changebot-provider>

Your App Header

Click the badge above to see product updates!

The drawer will slide in from the right with all updates.
