What is this? This page demonstrates how to completely override the built-in theme system with your own custom colors using CSS variables. No theme attributes needed!

Instead of using the built-in themes like catppuccin-mocha or dracula, you can define your own color scheme by setting CSS variables directly in your stylesheet.

How It Works

Simply add a <style> tag (or use your existing CSS) to define CSS variables on the component elements:

<style>
          /* Customize the badge - BRIGHT ORANGE! */
          changebot-badge {
          --cb-badge-bg: #ff6600; /* Bright orange background */
          --cb-badge-text: #ffffff; /* White text */
          --cb-badge-shadow: rgba(255, 102, 0, 0.5);
          --cb-badge-pulse: rgba(255, 102, 0, 0.8);
          }

          /* Customize the drawer - CYBERPUNK THEME! */
          changebot-panel {
          --cb-panel-bg: #0a0014; /* Deep purple-black background */
          --cb-panel-surface: #1a0033; /* Purple card backgrounds */
          --cb-panel-text: #00ffff; /* Cyan text */
          --cb-panel-accent: #ff6600; /* Bright orange titles */
          --cb-panel-link: #00ff00; /* Lime green links */
          --cb-panel-meta: #00ffff; /* Cyan dates */
          --cb-panel-border: #ff00ff; /* Magenta borders */
          /* ... and more! */
          }
          </style>

          <!-- Then use components WITHOUT theme attributes -->
          <changebot-provider slug="your-team" />
          <changebot-badge></changebot-badge>
          <changebot-panel display-mode="drawer-right"></changebot-panel>

Pro Tips

Light/Dark Mode Support

Use media queries to automatically switch between light and dark themes:

@media (prefers-color-scheme: light) {
          changebot-panel {
          --cb-panel-bg: #ffffff;
          --cb-panel-text: #1a1d29;
          /* ... your light theme colors ... */
          }
          }

          @media (prefers-color-scheme: dark) {
          changebot-panel {
          --cb-panel-bg: #1a1d29;
          --cb-panel-text: #e8eaed;
          /* ... your dark theme colors ... */
          }
          }

Match Your Brand

Set colors to match your brand's design system:

changebot-badge {
          --cb-badge-bg: var(--your-brand-primary);
          --cb-badge-text: var(--your-brand-text);
          }

          changebot-panel {
          --cb-panel-accent: var(--your-brand-primary);
          --cb-panel-link: var(--your-brand-link);
          }

Scope-Specific Themes

Apply different themes to different scopes using attribute selectors:

/* Admin scope - red theme */
          changebot-badge[scope="admin"] {
          --cb-badge-bg: #dc2626;
          }

          /* User scope - blue theme */
          changebot-badge[scope="user"] {
          --cb-badge-bg: #2563eb;
          }

Complete Example

Here's a full working example you can copy and paste:

<!DOCTYPE html>
          <html>
          <head>
          <script type="module" src="path/to/widgets.esm.js"></script>

          <style>
          changebot-badge {
          --cb-badge-bg: #ff6600; /* Bright orange */
          --cb-badge-text: #ffffff; /* White */
          }

          changebot-panel {
          --cb-panel-bg: #0a0014; /* Deep purple-black */
          --cb-panel-surface: #1a0033;/* Purple */
          --cb-panel-text: #00ffff; /* Cyan */
          --cb-panel-accent: #ff6600; /* Bright orange */
          --cb-panel-link: #00ff00; /* Lime green */
          --cb-panel-border: #ff00ff; /* Magenta */
          }
          </style>
          </head>
          <body>
          <changebot-provider slug="your-team" />

          <!-- In your header -->
          <changebot-badge></changebot-badge>

          <!-- Drawer (hidden until badge is clicked) -->
          <changebot-panel display-mode="drawer-right"></changebot-panel>
          </body>
          </html>
